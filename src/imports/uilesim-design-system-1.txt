Сделай полный “каркас фронта” (Figma design system + экраны) для Uilesim так, чтобы после дизайна оставалась только интеграция с Django DRF API. Работай внутри текущего Figma файла, не ломай уже готовые Header/Navbar + Home (Hero) и выбранную палитру: primary CTA/кнопки Ink (тёмные), розовый — только акцент. Стиль: premium minimal, светлые перламутровые фоны, мягкие тени, много воздуха. Никакого фиолетового.

Создай Figma Pages:
1) “App / Customer” (все страницы)
2) “Components / UI Kit” (все компоненты + варианты)
