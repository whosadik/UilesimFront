Создай дизайн главной страницы e-commerce (косметика/уход/ароматы) для бренда Uilesim. Нужны Navbar + Hero section (первый экран) в стиле premium minimal: чисто, дорого, много воздуха, аккуратная типографика, без кричащих цветов. Вдохновляйся общими принципами современных beauty-магазинов, но не повторяй композицию, шрифты и сетку сайта goldapple.kz.

1) Артборды и сетка

Сделай 3 фрейма:

Desktop: 1440×900

Tablet: 834×960

Mobile: 390×844

Для Desktop используй 12 колонок, max width контента 1160, gutters 24, поля 140 (или разумно близко).

Базовый spacing: 8px grid.

2) Визуальный стиль (чтобы не было копии)

Используй тёмный “cinematic” фон с мягкими градиентами + лёгким шумом/зерном.

Добавь непрямой акцент (например, сиренево-розовый / перламутровый) только в мелких деталях: линия, pill, кнопка.

Обязательно сделай hero НЕ как “текст по центру на фоне как у референса”. Сделай асимметричную композицию:

слева — текстовый блок,

справа — отдельная “promo card” (стеклянная карточка / blur), либо вертикальный блок со спецпредложением.

В hero добавь “мягкую кинематографичность”: motion blur на фоне, но текст должен быть читабельным.

3) Цвета (переменные)

Сформируй палитру и используй её в компонентах:

Background: #070A12

Surface glass: rgba(255,255,255,0.06) / rgba(255,255,255,0.10)

Border: rgba(255,255,255,0.14)

Text primary: #F8FAFC

Text secondary: rgba(248,250,252,0.72)

Accent soft: #D7B7FF

Accent strong: #7C3AED
Добавь легкий шум/грейн на фон (очень тонко).

4) Типографика

Шрифт: Inter (или Manrope, если Inter недоступен).

Desktop:

H1: 52–56 / 110% / weight 700–800

Body: 16 / 150% / 400–500

Small: 12–13 / 140%

Сделай оптические отступы и аккуратный letter-spacing для заголовка (-1% примерно).

5) Navbar (структура)

Сверху сделай двухуровневый хедер, но не как у референса по расположению:

Topbar (тонкая полоса):

Слева: язык “Қаз” (chip), город “Астана · Мангилик Ел, 53/1” (chip)

Справа: ссылки: “Доставка”, “Магазины”, “Поддержка”

Высота topbar: 40–44px

Стиль chip: pill, тонкий бордер, слегка прозрачный фон

Main navbar:

Слева: логотип Ü + Uilesim (mark в квадрате с градиентом + слово)

По центру: меню: “Каталог”, “Бренды”, “Новинки”, “Акции”, “Подарочные карты”

Справа: иконки (Search, Heart, User, Cart) в квадратных кнопках 40×40

Navbar sticky визуально: полупрозрачный фон + blur

Состояния:

hover у пунктов меню: подсветка тонкой градиентной линией снизу

active у “Новинки” (если выбрано): чуть ярче + underline

Mobile navbar:

На 390 ширине меню уходит в бургер

При открытии: выезжающее меню (sheet) со списком ссылок

6) Hero section (главный экран)

Сделай hero высотой ~560–620px (desktop).

Фон hero:

Используй абстрактный “beauty promo” стиль: мягкие блики, motion blur, перламутр.

Дополнительно наложи:

radial gradient слева сверху (accent soft),

radial gradient справа снизу (accent strong),

затемнение снизу для читабельности текста.

НЕ используй идентичный постер/кадр как у референса; сделай либо абстракцию, либо новый визуал (например “перламутровые ленты/монеты/проценты” как тема скидок, но без прямого копирования).

Контент слева (текст):

Маленькие бейджи (pill):

“Новые поступления”

“Скидки до −50%”

Заголовок H1: “Новинки недели”

Подзаголовок: “Уход, косметика, ароматы и товары для дома — свежие релизы и лимитки в одном месте.”

CTA кнопки (рядом):

Primary: “Смотреть новинки”

Secondary/Ghost: “Открыть каталог”

Ниже 3 мини-стата (в маленьких карточках):

“Бонусы — до 7%”

“Доставка — день-в-день”

“Подбор — по типу кожи”

Правая часть (отличие от референса):

Сделай promo card (glassmorphism):

Заголовок: “Персональный оффер”

Текст: “Ответь на 5 вопросов и получи скидку/множитель баллов под твои предпочтения.”

Кнопка: “Получить оффер”

Подсказка: “Займёт ~30 секунд”

Под карточкой: индикатор слайдов (dots): 3 точки, первая активна (активная — вытянутая капсула)

7) Компоненты и Auto Layout

Все блоки должны быть собраны через Auto Layout.

Создай компоненты:

Button (primary, ghost)

IconButton

Chip

Badge

NavLink

PromoCard

StatMini

Dots indicator

Для компонентов пропиши:

padding

radius (14–18)

border

hover/pressed варианты (создай variant states)

8) Accessibility / читаемость

Контраст текста на фоне должен быть достаточным (не теряй заголовок).

Добавь затемняющий overlay под текстом.

Все кликабельные элементы минимум 40×40 на mobile.

9) Имена слоёв (важно)

Именуй слои аккуратно:

Header/Topbar

Header/Navbar

Hero/Background

Hero/Content

Hero/PromoCard

Components/…

На выходе должны быть 3 готовых фрейма (desktop/tablet/mobile) с одинаковой логикой, адаптированные:

На tablet promo card уходит ниже текста или становится менее широкой.

На mobile: заголовок меньше, CTA в колонку, promo card ниже, точки по центру.
